## VSCode 插件名称

- 在 VSCode 扩展中搜索 `vscode-extension-vue`

## 特性

| 命令           | 描述                     |
| -------------- | ------------------------ |
| init-vue2      | vue2 初始化页面          |
| init-uniapp    | uniapp 初始化页面        |
| init-setup     | vue3 setup 初始化页面    |
| init-setup-ts  | vue3 setup ts 初始化页面 |
| init-list      | 初始化列表               |
| init-form      | 初始化表单               |
| init-component | 初始化组件               |
